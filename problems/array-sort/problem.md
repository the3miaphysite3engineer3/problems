Sort an array of integers in ascending order.

## Input:

- Array $a$ of integers of size $n$

## Output:

- The same array $a$ sorted in ascending order stored in $b$

## Notes:

- The input array $a$ is stored in row-major order in memory

## Test Case Sizes

- size = 16384
- size = 32768
- size = 65536
- size = 131072
- size = 262144
