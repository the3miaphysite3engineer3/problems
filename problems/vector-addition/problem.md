Perform element-wise addition of two vectors:
$$
c_i = a_i + b_i
$$

## Input
- Vectors $a$ and $b$ of length $N$

## Output
- Vector $c$ of length $N$ containing the element-wise sum

## Test Case Sizes

- n = 2^20
- n = 2^22
- n = 2^23
- n = 2^25
- n = 2^26
- n = 2^29
- n = 2^30
